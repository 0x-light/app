# Nimi App

Nimi app codebase.

# Development

## Setup

Clone

```bash
git clone https://github.com/nimi-app/nimi-frontend
cd nimi-frontend
```

Install dependencies

```bash
npm i
```

Some components, like contract factories and types, must be generated using:

```bash
npm run contracts:codegen
```

## Start

To start the app: 

```bash
npm run start
```
